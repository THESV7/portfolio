# Smit Vadhel - Frontend Web Developer Portfolio

Welcome to my personal developer portfolio! This project showcases my skills, experience, and the projects I've built as a Frontend Web Developer based in Mumbai.

## 🚀 Features
- **Modern & Sleek UI:** Built with a focus on clean design, typography, and fluid user experience.
- **Dark Mode Support:** Seamlessly toggle between light and dark themes with optimized color palettes and dynamic logo inversions.
- **Smooth Animations:** Includes smooth scrolling, custom cursors, and entrance animations utilizing Framer Motion.
- **Responsive Layout:** fully optimized for all devices ranging from mobile to large desktop screens.

## 💻 Tech Stack
- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Library:** [React](https://reactjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)

## 📂 Featured Projects
- **React Nex NPM Component Library:** Highly customizable UI components for React and Next.js, featuring single-component installation to minimize bundle size.
- **CineFind Movie Discovery App:** A responsive movie discovery platform utilizing React.js, Tailwind CSS, and the TMDB API.
- **TestFunda AI Mock Analyzer:** A Chrome Extension powered by Gemini AI to extract and analyze mock exam data into performance reports.
- **E-Commerce Website:** A robust e-commerce solution with a secure shopping cart and filtered product listings.
- **Nike Shoes UI:** A modern landing page to showcase modern UI/UX design principles.
- **Dashboard UI:** An intuitive admin dashboard for managing complex data sets and user activities.

## 🛠️ Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/THESV7/portfolio.git
   ```

2. Navigate into the directory:
   ```bash
   cd portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
